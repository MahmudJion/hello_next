# hello_next
A nextjs application with the example of data fetching from axios, components are using in pages and async function calling.

### Version
1.1.0

### Installation

Install the dependencies

```sh
$ npm install
```

### Development
To run in the development mood

```sh
$ npm run dev
```

### Build
Compile and build

```sh
$ npm run build
```
### Serve
To serve in the browser

```sh
$ npm start
```
